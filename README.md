# 📬 保研套磁统计工具

一个本地运行的网易邮箱保研套磁邮件统计工具。通过只读方式访问邮箱，自动识别套磁邮件、重建会话、判断导师回复状态，生成完整的统计报表。

## ⚠️ 数据隐私声明

- **只读访问**：软件通过 IMAP 只读模式访问邮箱，不会发送、删除、移动或修改任何邮件
- **本地存储**：所有邮件数据保存在本机 SQLite 数据库中，不上传到任何服务器
- **授权码安全**：默认不保存客户端授权码，可选使用系统 keyring 安全存储
- **AI 分析可选**：启用 OpenAI 分析时，部分邮件文本会发送给 API（需用户明确同意）
- **建议**：使用专门的客户端授权码，用完后可随时在网易邮箱设置中撤销

## 🖥️ Windows 安装步骤

### 前提条件
- Python 3.11 或以上版本（[下载地址](https://www.python.org/downloads/)）
- 安装时勾选 "Add Python to PATH"

### 方式一：使用安装脚本（推荐）
- 双击运行 **`setup_windows.bat`** (最简单)
- 或者右键 `setup_windows.ps1` -> 使用 PowerShell 运行

### 方式二：手动安装
```powershell
# 1. 创建虚拟环境
python -m venv .venv

# 2. 激活虚拟环境
.venv\Scripts\Activate.ps1

# 3. 安装依赖
pip install -r requirements.txt
```

## 🚀 启动方式

### 方式一：使用启动脚本
- 双击运行 **`run_windows.bat`** (最简单)
- 或者右键 `run_windows.ps1` -> 使用 PowerShell 运行

### 方式二：手动启动
```powershell
.venv\Scripts\Activate.ps1
streamlit run app.py
```

浏览器会自动打开 http://localhost:8501

## 📧 如何在网易邮箱中开启 IMAP

1. 登录网易邮箱网页版（mail.163.com）
2. 进入 **设置** → **POP3/SMTP/IMAP**
3. 开启 **IMAP/SMTP服务**
4. 按提示获取 **客户端授权码**（非登录密码）

> ⚠️ **为什么使用客户端授权码而非登录密码？**
> - 授权码是专门用于第三方客户端的临时密码
> - 可以随时在网易邮箱设置中单独撤销，不影响网页登录
> - 比直接使用登录密码更安全

## 📖 使用指南

### 1. 测试邮箱连接
- 打开"邮箱连接"页面
- 填写邮箱地址和客户端授权码
- 点击"测试连接"
- 确认显示连接成功和文件夹列表

### 2. 执行首次完整扫描
- 在"文件夹选择"页面确认要扫描的文件夹
- 进入"扫描与同步"页面
- 点击"开始扫描"
- 等待扫描完成（首次可能需要几分钟）

### 3. 增量同步
- 再次点击"开始扫描"
- 系统自动识别新邮件，只下载未同步的内容
- 基于 UID 和 UIDVALIDITY 实现增量同步

### 4. 使用本地规则模式
- 核心功能完全在本地运行，不需要任何 API
- 通过关键词匹配识别套磁邮件和回复内容
- 自动生成招生意向判断和行动建议

### 5. 可选启用 OpenAI 分析
- 进入"AI 分析设置"页面
- 设置环境变量 `OPENAI_API_KEY`（或在界面中输入）
- 可选设置 `OPENAI_MODEL`（默认 gpt-4o）
- 勾选"同意发送邮件内容给外部 AI"
- 选择要分析的导师，点击"开始分析"

> ⚠️ **ChatGPT/Codex 订阅 ≠ OpenAI API 额度**
> - ChatGPT Plus 订阅费不包含 API 调用额度
> - 需要在 [platform.openai.com](https://platform.openai.com) 单独充值
> - 首次注册有免费额度，但可能已过期

### 6. 导出 Excel
- 进入"导出与数据管理"页面
- 选择导出格式（Excel/CSV/JSON）
- 点击导出
- Excel 包含 6 个工作表：导师总表、汇总统计、邮件明细、待人工复核、扫描错误、配置说明

### 7. 处理"无法确认"记录
- 进入"人工复核"页面
- 查看需要确认的项目
- 可以手动修改导师信息、回复状态、招生意向
- 手动修改在重新扫描后不会被覆盖

### 8. 撤销网易客户端授权码
1. 登录网易邮箱网页版
2. 进入 **设置** → **POP3/SMTP/IMAP**
3. 点击 **重置授权码** 或关闭 IMAP 服务

## ❓ 常见错误解决

| 错误 | 解决方法 |
|------|---------|
| IMAP 连接失败 | 确认已开启网易邮箱的 IMAP 服务 |
| 认证失败 | 使用客户端授权码，而非登录密码 |
| SSL 错误 | 确认端口为 993 且 SSL 已开启 |
| 连接超时 | 检查网络连接，或尝试增加超时时间 |
| 文件夹无法读取 | 可能是网易服务器限制，稍后重试 |
| Excel 被占用 | 关闭正在打开的 Excel 文件后重试 |
| API Key 无效 | 检查 OPENAI_API_KEY 环境变量格式 |
| 扫描结果为零 | 确认选择了正确的文件夹（已发送、收件箱）|
